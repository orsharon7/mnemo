Why Mnemo?

The clipboard is a tool, not a product.

Modern clipboard managers ship sync, sharing, integrations, and a sign-in screen. Mnemo doesn't. It's the small, native, keyboard-first utility that just remembers what you copied so you can paste it again — the Alfred clipboard experience as a standalone app.

Local by default. Private by design.

Mnemo runs with zero permissions out of the box. The only optional permission is Accessibility, and only if you opt into auto-paste. Concealed pasteboard items and likely secrets are filtered before they ever hit disk.

Fast where it matters.

Press the hotkey, type a few characters, hit ⏎. Mnemo copies, dismisses, and refocuses the previous app in one motion. Optional auto-paste turns that into a single keystroke.

Native, minimal, free.

Built with Swift and AppKit. No Electron, no background services, no subscription. On-device semantic search uses Apple's NLEmbedding — no API key, no network call, ever.
